Пояснення послідовності дій
Користувач через інтерфейс (UI) надсилає запит «Взяти книгу».

LibraryUI передає запит у BookService, який перевіряє доступність книги.

BookService звертається до BookRepository, щоб отримати статус книги.

Якщо книга доступна, ReservationService створює бронювання.

ReservationRepository зберігає дані про бронювання у базі.

Система повертає підтвердження користувачу: «Книга заброньована».

Sequence Diagram деталізує, як саме цей сценарій реалізується через взаємодію між об’єктами.
