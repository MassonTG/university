Пояснення станів і переходів
Доступна Книга знаходиться у бібліотеці й може бути заброньована або видана.

Перехід: користувач бронює → стан «Заброньована».

Перехід: адміністратор видає напряму → стан «Видана».

Заброньована Книга зарезервована для конкретного користувача.

Перехід: користувач приходить і отримує книгу → «Видана».

Перехід: бронювання скасовано → повернення у «Доступна».

Видана Книга знаходиться у користувача.

Перехід: користувач повертає книгу → «Повернена».

Повернена Книга повернулася у бібліотеку, але ще не перевірена/не внесена у систему.

Перехід: після обробки бібліотекарем → «Доступна».
